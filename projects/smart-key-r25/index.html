Smart Key R25

Sistem keamanan motor Yamaha R25 berbasis ESP32 dan Bluetooth. Motor hanya bisa dinyalakan jika HP pemilik terdaftar terhubung via Bluetooth, dengan fitur auto-lock otomatis, PIN pairing, MAC address filter, dan PIN backup.

Fitur Utama

Auto Unlock

Motor otomatis terbuka saat HP pemilik yang terdaftar connect Bluetooth ke ESP32.

Auto Lock (10 detik)

Motor otomatis terkunci 10 detik setelah HP disconnect dari Bluetooth.

PIN Pairing Bluetooth

HP harus memasukkan PIN 6 digit saat pairing pertama kali. Tanpa PIN, tidak bisa connect.

MAC Address Filter

Hanya HP yang MAC address-nya terdaftar yang bisa unlock. HP lain otomatis ditolak dan di-disconnect.

PIN Backup

Jika HP utama tidak tersedia, bisa unlock menggunakan PIN via Bluetooth Serial Terminal.

Serial Command

Kontrol penuh via aplikasi Serial Bluetooth Terminal (LOCK, UNLOCK, STATUS, SET_PIN).

Persistent Storage

PIN dan daftar HP tersimpan di flash memory ESP32. Tidak hilang meski ESP32 restart atau mati.

Anti Tekor Aki

Dilengkapi tombol on/off pada jalur power buck converter untuk memutus daya saat motor tidak digunakan.

Source Code (Preview)

Smart_Key_R25_BT_Only.ino 
#include "sdkconfig.h"
#include <Preferences.h>
#include <BluetoothSerial.h>

#define RELAY_PIN 26

const char* BT_NAME = "R25_SmartKey";
const char* BT_PIN_CODE = "******";  // PIN pairing 6 digit
const int MAX_HP = 3;

BluetoothSerial SerialBT;
Preferences prefs;

String registeredMAC[MAX_HP];  // MAC Address HP terdaftar
int hpCount = 0;

void setup() {
  Serial.begin(115200);
  pinMode(RELAY_PIN, OUTPUT);
  digitalWrite(RELAY_PIN, LOW);

  SerialBT.begin(BT_NAME);
  SerialBT.setPin(BT_PIN_CODE, strlen(BT_PIN_CODE));
  SerialBT.register_callback(btCallback);

  loadData();
  Serial.println("System ready!");
}

void btCallback(esp_spp_cb_event_t event, ...) {
  if (event == ESP_SPP_SRV_OPEN_EVT) {
    // Cek MAC address HP
    if (isMACRegistered(currentMAC)) {
      unlockRelay();   // HP terdaftar - UNLOCK
    } else {
      SerialBT.disconnect();  // HP asing - TOLAK
    }
  }
  if (event == ESP_SPP_CLOSE_EVT) {
    // Auto LOCK setelah 10 detik
  }
}
